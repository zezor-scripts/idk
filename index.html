Lua Scripting Fundamentals

A Quick Reference Guide for Roblox Developers

1. Variables & Data Types

Variables are containers for storing information. The local keyword makes them private to the current script or function, which is best practice.

Key Data Types

  • string: Text (e.g., "Hello World")
  • number: Whole or decimal numbers (e.g., 5, 3.14)
  • boolean: True or False (e.g., true, false)
  • Instance: A reference to an object in the Roblox game world (e.g., a Part, a Player, a Script)
local part = game.Workspace.MyPart -- Instance (object reference)
local playerSpeed = 16 -- Number
local message = "Welcome to the game!" -- String
local isDay = true -- Boolean

2. Conditional Logic (if/then/else)

Control the flow of your code by executing blocks of instructions only when a condition is met.

local playerScore = 100

if playerScore >= 100 then
    -- Code runs ONLY if the condition is TRUE
    print("You unlocked a new level!")
elseif playerScore > 50 then
    print("Great progress!")
else
    print("Keep trying!")
end

3. Functions

Functions are blocks of code that perform a specific task. They help organize your script and avoid repeating code.

-- Define the function with two parameters (num1 and num2)
local function addNumbers(num1, num2)
    local sum = num1 + num2
    return sum -- Send the result back
end

-- Calling the function and storing the result
local result = addNumbers(5, 12)
print(result) -- Output: 17

4. Events (.Touched & :Connect())

Events allow your code to react to actions that happen in the game, such as a player touching a part or clicking a button.

local part = game.Workspace.KillBrick

local function onPartTouched(otherPart)
    -- Check if the object that touched the brick is part of a Humanoid
    local humanoid = otherPart.Parent:FindFirstChild("Humanoid")

    if humanoid then
        humanoid.Health = 0 -- Kill the player
    end
end

-- Connect the function to the 'Touched' event of the part
part.Touched:Connect(onPartTouched)

5. Loops (for & while)

Loops repeat a block of code multiple times. The most common use is the while true do loop for continuous scripts.
